Terms

Version 4.2.1 · March 2026

1. Definitions

CODE 98 refers to the proprietary queue-based settlement engine. Participant means any individual or entity that initiates a consideration transfer. Queue means a chronological container where consideration events are ordered by receipt timestamp. Denomination means the exact amount remitted.

2. Queue Formation

Upon validation, the participant is appended to the tail of the queue corresponding to the exact amount. Each denomination operates independently — no cross-denomination aggregation is permitted.

3. Settlement

A disbursement for the head of any queue is triggered upon detection of a subsequent enqueuing within the same denomination. The instruction comprises return consideration to the original participant and administrative fee to the designated account.

4. Denomination Integrity

The system enforces strict denomination integrity: each amount pairs only with the same amount.

5. Administrative Fee

The administrative fee is fixed per denomination as published.

6. Disclaimer

The CODE 98 system is provided on an "as is" basis. CODE 98 shall not be liable for delays in disbursement or inability to access queues.

7. Governing Law

This Agreement shall be governed by the laws of the Republic of Kenya.
